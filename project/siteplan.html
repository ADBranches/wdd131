RealityMix Site Plan

Site Name

RealityMix - Blending digital and physical worlds.

Mobile View

  • Header: Collapsible menu (☰) + logo
  • AR Demo: File upload → Preview (vertical stack)
  • Quiz: Full-width cards with radio buttons

Desktop View

  • Header: Logo + inline navigation
  • Main: 2-column grid (controls | preview)
  • Sidebar: Fixed quiz widget
### **1. Mobile Wireframe** **Mermaid Code:** ```mermaid flowchart TB subgraph Mobile A[☰ Menu] --> B[RealityMix Logo] C[Hero: Try AR Demo] --> D[Upload Photo Button] D --> E[AR Preview Pane] F[Quiz: Q1] --> G[Q2] G --> H[Result] end ``` **Text Description:** - **Top Bar:** Hamburger menu + logo - **Hero Section:** Full-width "Try AR Demo" button - **AR Tool:** File upload → Preview pane (stacked vertically) - **Quiz:** Accordion-style questions --- ### **2.
